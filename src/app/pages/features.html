WHAT IS CODEREADY TOOLCHAIN?

CodeReady Toolchain provides an integrated approach to DevOps with all the tools a team needs to analyze, plan, create, and deploy services. Main features include:

  • One-click Linux container management
  • Utilizes many open source projects like fabric8, Eclipse Che, and OpenShift Online
  • Machine learning and full stack analysis
  • Built-in planning tools

MAKE MORE CONFIDENT TECHNOLOGY DECISIONS

CodeReady Toolchain provides actionable recommendations and insights around the software components you choose to use in your applications. Built-in analytics and machine learning offer the following benefits:

  • Get alerts about security vulnerabilities or dependency issues in your source code and your build reports
  • Use full stack analysis to help you choose the right packages and libraries
  • Get tangible information from sources like GitHub and Maven to support your decision-making

CREATE CONTAINERIZED DEV, TEST, AND STAGE ENVIRONMENTS WITH EASE

CodeReady Toolchain creates environments hosted on OpenShift Online, providing the following advantages:

  • Integrated Jenkins Pipelines plugins
  • Flexibility to have simple automated build pipelines or add manual promotions
  • Pipeline definitions written in Groovy DSL

MINIMIZE TIME & EFFORT TO BUILD & MAINTAIN A DEVELOPMENT TOOL CHAIN

Use the workspace management capabilities of Eclipse Che to automatically create containerized development environments.

  • Code, edit, and debug with Eclipse Che
  • Review and analyze with the dashboard and reporting tools
  • Reduce your build and maintenance efforts
